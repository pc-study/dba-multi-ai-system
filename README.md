# DBA Multi-AI System

这是一个独立的开源项目，核心代码在 [`dba-team`](./dba-team) 目录下。

## 项目简介

DBA多AI协作系统 - 基于 OpenClaw 的数据库专家团队

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/pc-study/dba-multi-ai-system.git

# 进入目录
cd dba-multi-ai-system

# 查看dba-team目录
ls dba-team/
```

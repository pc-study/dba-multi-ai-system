# Contributing to DBA Multi-AI System

Thank you for your interest in contributing!

## How to Contribute

### 1. Add a New Expert

1. Create a new prompt file in `prompts/`
   ```bash
   # Example: prompts/newdb-expert.md
   # Follow the pattern of existing expert prompts
   ```

2. Update `SKILL.md` to add task routing rules

3. (Optional) Add documentation in `references/`

### 2. Improve Existing Prompts

- Keep prompts concise and actionable
- Include practical examples
- Maintain consistent format

### 3. Report Issues

- Database-specific questions not handled well
- Knowledge base links that are outdated
- Suggestions for new features

## Coding Standards

- Use clear, descriptive names
- Include comments for complex logic
- Follow existing file structure

## Commit Messages

Use clear commit messages:
- `feat: add TiDB expert`
- `fix: update Oracle documentation links`
- `docs: improve README`

## Questions?

Open an issue for discussion before submitting large changes.
